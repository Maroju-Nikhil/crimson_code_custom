Iot Gateway
CIT IoT gateway can be used to collect information from various IoT devices and make it available to apps, so that things can get managed through apps. Also it helps to create smartness among the devices. The IoT gateway software is a software solution that bridges the semantic gaps between the raw sensor data and the information context that is interested by high-level applications. The connection from end units to Internet or cloud is established at an edge system which serves as communication gateway to bridge local networks with cloud systems.

Features:

  • Provision to interface with end nodes by using BT/BLE, ZigBee, Wi-Fi, Ethernet
  • Web interface to discover, Configuring of end nodes
  • Interfacing to Clouds such as MQTT and Firebase
  • Management of Gateway and End Nodes
  • Alerts and Notifications
  • Device to device communication in offline mode
  • Secured communication

FunLearn
FunLearn is an electronic gadget developed based on LPC2148 to help students improve their competitive skills. This device consists of a SD card, LCD panel and navigation keys . All the tests related data is stored in SD card and this devices comes with an option to set password for security.
Fun Learn maintains 3 levels of menus where it displays Types of tests, Subjects and the questions in respective menus. Navigating through these menus, the student can attempt a test which consists of multiple choice questions. Once the test is completed, the result is displayed. There is an option to store the result for future reference which is stored on SD card.
  • Embedded gadget
  • Learning aid for students
  • Controller:LPC2148(ARM7)
  • 2GB SD card with fat32 for storing encrypted datades
  • LCD for displaying
  • Keypad for user interaction
  • Rechargeable battery

CSVM

Crimson Storage Validation Manager (CSVM)

Crimson Storage Validation Manager (CSVM) is a cloud based test tool. The user will be able to access his/her test lab where the SSD & NVMe devices are connected to test machines, from anywhere globally.

User can have complete control over the validation process and can start, stop, monitor, and collect the reports of validation from anywhere

CSVM is capable of testing the drives for the DVT, QA and Production purposes.

The suite supported interfaces are:
  • SATA
  • NVMe
  • SAS

AgriHealth

Food production will have to be increased in order to meet the demand caused by increase in world population. The increase in food production can occur by achieving better yields in the current cultivated areas.

Nitrogen (N) is an essential plant nutrient and, next to water. The agricultural input that resulted in greater increase in crop productivity is nitrogen (N) fertilizer. Given the importance of N fertilizer to increase crop productivity and also the environmental concerns from its over-application, management practices that optimize the use of N, increasing its use efficiency and decreasing its losses, are warranted.

Crimson Innovative Technologies (CIT) proposes a AgriHealth which is integrated to cloud. Some plants may require more fertilizers and some may require less compared to other parts of the crop. AgriHealth enables the end user to apply fertilizers as per plant requirement. Future crops can be planned based on statistics of past crop’s history obtained from the cloud.

Advantages
  • Reduce input fertilizer costs.
  • Allows fertilizers to be adjusted during the growing season to meet plant requuirements
  • Only apply fertilizer to areas that need it.
  • Collect data anytime
  • Past history and statistics of the crop
  • Deduplication for eliminating duplicate data
